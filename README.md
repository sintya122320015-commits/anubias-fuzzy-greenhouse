# Rancang Bangun Sistem Kontrol Suhu dan Kelembapan Anubias sp Metode Emersed

Penelitian ini mengembangkan sistem kontrol suhu udara dan kelembapan udara menggunakan metode Fuzzy Mamdani pada mini greenhouse untuk budidaya Anubias sp metode emersed.

## Sistem yang Digunakan

- Mikrokontroler ESP32
- Sensor DHT22
- Sensor DS18B20
- Mist Maker
- Kipas DC
- Growlight Kandila S1000
- RTC DS3231

## Metode Kontrol

Metode kontrol yang digunakan adalah Fuzzy Logic Mamdani dengan input:

- Suhu udara
- Kelembapan udara

Output:

- PWM Kipas
- Mist Maker

## Parameter Lingkungan

- Suhu udara : 22 – 28 °C
- Kelembapan udara : 80 – 90 %
